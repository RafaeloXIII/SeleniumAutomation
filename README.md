This repository contains an ETL automation project built with Selenium to streamline data collection from web-based systems.

What it does

Automates navigation and data extraction from target websites

Downloads XLSX files automatically

Processes and loads extracted data into a database

Keeps datasets current for BI reporting and dashboards

Why it was built

Manual data collection was time-consuming and error-prone.
This automation was created to improve reliability, reduce repetitive operational work, and increase data availability for analytics teams.

Impact

Reduced direct manual interaction by ~85%

Increased data refresh consistency

Accelerated report and dashboard readiness

Typical use cases

Scheduled extraction of operational spreadsheets

Centralized database ingestion for analytics

Data pipeline support for internal BI tools
